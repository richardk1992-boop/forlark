        使用说明
        
    
          
  1. 在飞书开放平台创建企业自建应用并获取 App ID 和 App Secret
    2. 
          
  2. 为应用添加"查看云文档内容"权限 (docx:document:readonly)
    3. 
          
  3. 部署OAuth回调页面：
            
      
              
    • 在项目目录找到 callback.html 文件
      • 
              
    • 创建一个GitHub仓库并上传 callback.html
      • 
              
    • 在GitHub仓库设置中启用 GitHub Pages
      • 
              
    • 你的回调URL将是: https://你的用户名.github.io/仓库名/callback.html
              
    • 将此URL添加到飞书开放平台的重定向URL列表中
      • 
            
    
          
    4. 
          
  4. 在 background.js 中找到 callbackUrl 变量，替换为你的实际 GitHub Pages URL
    5. 
          
  5. 重新加载插件以应用更改
    6. 
          
  6. 发布应用或开启测试版本（重要：权限需要审批通过）
    7. 
          
  7. 在上方输入框中填入 App ID 和 App Secret 并保存
    8. 
          
  8. 点击"用户登录授权"按钮，完成飞书账号授权
    9. 
          
  9. 打开任意飞书文档，点击"获取文档内容"按钮
    10. 
          
  10. 内容获取后可以复制或下载
    11. 
        

        

          ⚠️ 重要提示：
          
    
            
  • 应用类型必须是"企业自建应用"
    • 
            
  • 权限审批需要企业管理员批准
    • 
            
  • 用户授权可访问私密文档，应用身份只能访问公开文档
    • 
            
  • 区域必须匹配：
              
      
                
    • 中国版文档(feishu.cn) → 应用在 open.feishu.cn 创建
      • 
                
    • 国际/新加坡版文档(larksuite.com/larkoffice.com) → 应用在 open.larksuite.com 创建
      •
